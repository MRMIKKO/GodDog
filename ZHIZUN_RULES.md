# 天九至尊规则说明 🎴👑

## 特殊组合：至尊

### 1. 母至尊（无敌组合）🔥
**组成**：三点 + 六点

**特点**：
- ✅ **绝对无敌**：先出母至尊，没有任何牌能打败
- 💰 **收取积分**：母至尊获胜时，可以收取所有其他玩家的积分（以栋数计算）
- 🎯 **先出为大**：只有先手出母至尊才有此特权

**示例**：
```
玩家A先出：三点 + 六点（母至尊）
其他玩家：无论出什么牌都无法打败
结果：玩家A获胜，收取其他3位玩家的所有积分
```

---

### 2. 文至尊（双鼻屎六）👑
**组成**：两张鼻屎六（rank=11的文牌对子）

**特点**：
- ✅ **碾压文牌**：先出文至尊，所有文牌对子都打不过（包括双天！）
- ⚠️ **唯一弱点**：只有高脚七对子能打败文至尊
- 💰 **收取积分**：文至尊获胜时，收取其他玩家的积分
- ⚡ **被打败的惩罚**：如果被高脚七打败，要支付积分给对方

**示例1（文至尊获胜）**：
```
玩家A先出：双鼻屎六（文至尊）
玩家B出：双天（最大的文牌对子）
结果：玩家A的文至尊依然获胜！收取积分
```

**示例2（文至尊被打败）**：
```
玩家A先出：双鼻屎六（文至尊）
玩家B出：双高脚七
结果：玩家B获胜！玩家A要支付手上的积分给玩家B
```

## 至尊规则总结

### 牌力排序（两张牌的情况）
1. **母至尊**（三点+六点）- 绝对最大
2. **文至尊**（双鼻屎六）vs **高脚七对子** - 高脚七胜
3. **文至尊**（双鼻屎六）- 打败所有其他文牌对子
4. **双天** - 普通文牌中最大
5. **双地** - 普通文牌第二大
6. ...其他文牌对子...

### 积分系统
- 每赢一栋 = +1栋数
- 栋数可以作为积分基础
- 至尊获胜时收取其他玩家的积分
- 文至尊被高脚七打败时要支付积分

### 战术建议

#### 持有母至尊时：
- 💡 **保存实力**：等到关键时刻再出，确保最大收益
- 💡 **观察积分**：等其他玩家积累栋数后再出，收益更大

#### 持有文至尊时：
- ⚠️ **小心高脚七**：注意观察是否有人持有高脚七对子
- 💡 **适时出击**：在确认没有高脚七威胁时果断出手
- ⚠️ **风险管理**：如果积分较高，要谨慎使用，避免被高脚七打败损失惨重

#### 持有高脚七对子时：
- 🎯 **克制文至尊**：这是唯一能打败文至尊的牌
- 💡 **伺机而动**：等待对手出文至尊时再使用
- 💰 **高额回报**：打败文至尊可以获得对方的所有积分

## 游戏胜利条件

原本的规则是"最后一栋定胜负"，加入至尊和积分系统后：

**最终胜者**：游戏结束时**积分最高**的玩家获胜

这样既保留了栋数的重要性（栋数=积分基础），又增加了至尊带来的策略深度和刺激性！

---

## 注意事项

1. ✅ 至尊必须**先出**才有特殊效果
2. ✅ 母至尊 > 文至尊（如果母至尊先出）
3. ✅ 高脚七是文至尊的克星
4. ✅ 积分是动态的，可正可负
5. ✅ 观察其他玩家的牌，推测是否有至尊或克制牌
