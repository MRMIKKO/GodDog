# 天九至尊 🎴

一款基于广西天九规则的4人联机网页游戏。

## 游戏规则

- **玩家人数**: 4人
- **牌数**: 32张天九牌，每人8张
- **胜利条件**: **赢得最后一栋（最后一轮）的玩家获胜**
- **出牌方式**: 单张、对子或组合
- **跟牌规则**: 必须出相同数量的牌，且牌面更大，或选择过牌

## 天九牌组成

### 文牌（22张，11种对子）
1. 天牌 - 12点（最大）
2. 地牌 - 2点
3. 人牌 - 8点
4. 饿牌 - 4点
5. 梅花 - 10点
6. 长山 - 6点
7. 板凳 - 4点
8. 斧头 - 11点
9. 平峰 - 10点
10. 高脚七 - 7点
11. 鼻屎六 - 6点

### 武牌（10张，单张）
- 九点A、九点B
- 八点A、八点B
- 七点A、七点B
- 五点A、五点B
- 三点、六点

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动服务器

```bash
npm start
```

服务器将在 `http://localhost:3000` 启动

### 开始游戏

1. 在浏览器中打开 `http://localhost:3000`
2. 输入您的昵称
3. 输入房间号（或留空自动创建）
4. 点击"进入游戏"
5. **系统会自动添加3个机器人玩家，方便单人调试**
6. 点击"准备"
7. 游戏自动开始！

### 机器人功能 🤖

为了方便开发和调试，系统已内置智能机器人：
- 当第一个真人玩家加入房间时，自动添加3个机器人
- 机器人会自动准备
- 机器人会智能出牌和过牌
- 机器人出牌有1-2.5秒的思考延迟，模拟真实玩家
- 单人即可测试完整游戏流程

## 本地联机测试

### 方式一：同一台电脑测试
1. 启动服务器: `npm start`
2. 打开4个浏览器标签页
3. 每个标签页输入不同的昵称和相同的房间号
4. 全部准备后开始游戏

### 方式二：局域网联机
1. 在一台电脑上启动服务器: `npm start`
2. 查看本机IP地址:
   - macOS: 在终端输入 `ifconfig | grep "inet "`
   - Windows: 在命令提示符输入 `ipconfig`
3. 其他玩家在浏览器中访问 `http://[本机IP]:3000`
4. 输入房间号加入同一个房间

## 游戏操作

### 出牌
- 点击手牌选择要出的牌（选中的牌会上浮并高亮）
- 点击"出牌"按钮打出选中的牌

### 过牌
- 点击"过"按钮放弃本轮跟牌

### 整理手牌
- 点击"整理手牌"按钮按牌力大小重新排列

## 游戏策略

### 核心策略
- **保留实力**: 前面的栋不必拼命去赢
- **算牌**: 记住已出的大牌，推测对手手中的牌
- **控制节奏**: 适时赢下某一栋，获得主动出牌权
- **最后一搏**: 留好大牌，争取赢下最后一栋

### 牌型大小
- **对子 > 单张**
- **文牌对子**: 天 > 地 > 人 > 饿 > 梅 > 长山 > 板凳 > 斧头 > 平峰 > 高脚七 > 鼻屎六
- **武牌单张**: 按点数比较
